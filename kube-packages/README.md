# kube packages

addon 들이 위치하며 기본적으로는 helm chart 를 사용하도록 함  

## addon 설치 방법

addon 디렉토리에 install-helm-charts.sh 을 이용하거나 포함되어 있는 values.yaml 을 복사 수정하여 설치할 수 있다
